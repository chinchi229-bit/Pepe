## Descripción
Este es un ejemplo de implementación de RoboFut con ESP32. 
Muestra la estructura recomendada para subir tu código.

## Características Implementadas
- ✅ Control de 2 motores traseros
- ✅ Paleta giratoria con motor independiente
- ✅ Conectividad WiFi
- ✅ Sincronización con servidor web
- ✅ Sensores de línea (opcional)

## Cambios Principales
- Ajuste de velocidad de motores
- Optimización del consumo de energía
- Mejor respuesta de controles

## Cómo Usar Este Template

1. Copia esta carpeta: `cp -r team_example tu_equipo_nombre`
2. Modifica los archivos con tu código
3. Actualiza `config.json` con tus parámetros
4. Sube tu carpeta al repositorio

## Especificaciones Técnicas
- ESP32 DevKit v1
- Motores DC 180-300 RPM
- Fuente 5V 3A
- Drivers L298N
