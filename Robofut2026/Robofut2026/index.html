🐛 SOLUCIÓN DE PROBLEMAS

¿El juego no carga?
  • Verifica que el navegador sea moderno (Chrome, Firefox, Safari, Edge)
  • Asegúrate de tener WebGL habilitado
  • Abre la consola (F12) para ver errores
¿Los controles no funcionan?
  • Teclado: Click en la ventana del juego para asegurar el foco
  • Gamepad: Conecta ANTES de abrir el juego
  • ESP32: Verifica el Serial Monitor (115200 baud)
¿Bajo rendimiento/FPS bajo?
  • Cierra otras pestañas del navegador
  • Reinicia el navegador
  • Intenta en otro navegador
